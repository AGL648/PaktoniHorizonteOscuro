# Paktoni: Horizonte Oscuro
Example of an Isometric 3D Action Adventure Type of Game.

# Description
This is a Video Game Project which aims to create a 3D adventure game with an isometric camera view. You need to help our brave knight Robert pass through the levels by collecting keys, destroying robots and solving different puzzles that are around the levels.

# Features
- 3D Isometric View.
- Different Puzzles.
- A World made of Fantasy Characters.
- Lots of actions.

# Programs Used
- Unity 6
- C# (Visual Studio)

# Lessons
I've learned how to make a 3D adventure type of game, with different puzzles and bosses.
